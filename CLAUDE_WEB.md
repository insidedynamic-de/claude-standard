# WEB RULES (React + Axios)

- Use generated OpenAPI client only
- Single axios instance with interceptors
- React Query for data fetching

UI states required:
- Loading
- Empty
- Error

Status colors:
- Success = green
- Info = blue
- Warning = yellow
- Error = red
- Neutral = gray

Theme:
- Light / Dark / System

Auto logout:
- Token expiry required
